# STSJ Converter – Telegram **SESSION → SESSION+JSON**  
轻量级工具，将 Telegram 会话文件转换为与 Telegram Prime 等客户端兼容的 JSON 格式。

---

## ⚙️ 主要功能

| 功能 | 说明 |
|------|------|
| **SESSION → SESSION+JSON** | 将原生 Telegram 会话文件转换为可与 Telegram Prime 和其他兼容客户端使用的 JSON 格式。 |
| **Pyrogram 支持** | 同时接受 Pyrogram 风格的会话文件。 |
| **重新授权** | 需要时创建全新的会话。 |
| **冷启动转换** | 可在不连接账号的情况下将 SESSION+JSON 转换为 TDATA。 |
| **旧层兼容** | 选项可保持与旧 Telegram 层的兼容性或保存为新层。 |
| **会话清理** | 可选择断开账户所有其它活跃会话。 |
| **自定义重试 / 延迟** | 设置连接尝试次数及两次尝试间的等待时间。 |
| **多线程** | 并行执行多个转换任务，提升批量处理速度。 |
| **细粒度 JSON 控制** | 在写入文件前编辑设备、语言等字段——防止被封禁的重要手段。 |
| **结果分类** | 有效会话 → “valid”，无效会话 → “invalid” 文件夹。 |
| **日志记录** | 保存完整的控制台操作日志。 |
| **备份** | 在任何更改前复制原始会话文件。
